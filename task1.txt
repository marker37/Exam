Створення гілки:
переключення на основну гілку - git checkout main
створення нової гілки - git branch new-feature 
перехід на нову гілку - git checkout new-feature


Об'єднання гілок:
перехід на гілку, яку хочете об'єднати - git checkout main
об'єднання нілок - git marge new-feature


Використання гілок додають зручності, безпеки, простоти комунікації і спрщення спільної роботи
